# Folium Café — Website

A clean, modern, and fully responsive website built for a fictional local café
in Parkview, Johannesburg. Created as part of a freelance web development internship project.

---

## Project Overview

| Detail       | Info                              |
|--------------|-----------------------------------|
| Business     | Folium Café                       |
| Type         | Restaurant / Café                 |
| Location     | 14 Birch Lane, Parkview, JHB      |
| Style        | Modern & Minimal                  |
| Built with   | HTML · CSS · Vanilla JavaScript   |

---

## File Structure

```
folium-cafe/
│
├── index.html    ← Page structure (sections & content)
├── style.css     ← All visual styling
├── script.js     ← All interactivity & menu data
└── README.md     ← You are here
```

> All three files must stay in the **same folder** to work correctly.

---

## Sections

| Section   | Description                                           |
|-----------|-------------------------------------------------------|
| Navbar    | Fixed top bar with smooth-scroll links                |
| Hero      | Full-screen welcome with headline and stat cards      |
| About     | Origin story, brand values, and key metrics           |
| Menu      | 3-tab menu (Breakfast / Lunch / Drinks) — 18 items   |
| Visit     | Address, hours, contact info, and map placeholder     |
| Footer    | Logo, copyright, and quick links                      |



## Tech Stack

- **HTML5** — semantic elements (`<nav>`, `<section>`, `<dl>`, `<footer>`)
- **CSS3** — custom properties, CSS Grid, Flexbox, IntersectionObserver animations
- **Vanilla JavaScript** — no frameworks, no dependencies
- **Google Fonts** — Cormorant Garamond (display) + DM Sans (body)

---

## Internship Context

This project was built as part of a real-world freelance simulation:

- **Goal:** Build a professional website for a local business with no online presence
- **Pitch angle:** Help the owner attract customers, look credible, and grow online
- **Demo method:** Host on Netlify → share the live link with the owner
- **Lesson:** This is how freelancers and agencies land their first paid clients
